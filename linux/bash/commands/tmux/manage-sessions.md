# `tmux` — manage sessions

Create, list, attach, detach, and kill tmux sessions from the command line.

## Basic usage

```sh
tmux                        # start a new session with an autogenerated name
tmux new -s work            # start a new session named "work"
tmux ls                     # list all running tmux sessions
tmux attach -t work         # attach to the session named "work"
tmux a                      # attach to the last active session
tmux detach                 # detach from the current session (from inside tmux)
tmux kill-session -t work   # terminate the session named "work"
tmux kill-server            # terminate all sessions and the tmux server
```

| Command / Option | Description |
| --- | --- |
| `new -s <name>` | Create a new session with a name |
| `ls` | List active sessions |
| `attach -t <name>` | Attach to a specific session (short: `a -t`) |
| `kill-session -t <name>` | Terminate a specific session |
| `kill-server` | Kill all sessions and the tmux daemon |

## Keybindings (inside tmux)

By default, tmux commands are triggered by pressing the prefix key `Ctrl-b` (written as `C-b`), followed by a shortcut key:

| Keybinding | Action |
| --- | --- |
| `C-b d` | Detach from the current session (keeps it running in the background) |
| `C-b s` | Interactively select and switch to another session |
| `C-b $` | Rename the current session |
